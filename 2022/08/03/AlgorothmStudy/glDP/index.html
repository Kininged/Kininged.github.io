例题1

http://poj.org/problem?id=2096

题意：

一个系统中有 nnn 个子系统和 sss 个bug类型。

每天，该系统的某一个子系统中会出现一个bug ，bug的类型是等概率的，bug也是等概率地出现在每个子系统的。

问所有子系统都出现bug且所有的bug类型都出现的期望天数。

解法:

设 f[i][j]f[i][j]f[i][j] 表示从 nnn 个里选出 iii 个， 从 sss 个里选出 jjj 个，到达 f[n][m]f[n][m]f[n][m] 的期望步数。有f[n][m]=0f[n][m] = 0f[n][m]=0。
讨论当前出现bug的各个情况：
属于已经发现的 i,ji, ji,j, 概率为 p1=i/n∗j/sp_1 = i/n * j / sp1​=i/n∗j/s;
属于已经发现的 iii 个子系统，不属于已经发现的 jjj 个类型，概率为 p2=i/n∗(s−j)/sp_2=i /n*(s - j) / sp2​=i/n∗(s−j)/s;
不属于已经发现的 iii 个子系统，属于已经发现的 jjj 个类型，概率为p3=(n−i)/n∗j/sp_3=(n-i)/n * j / sp3​=(n−i)/n∗j/s;
均不属于，概率为 p4=(n−i)/n∗(s−j)/sp_4=(n - i) / n * (s - j) / sp4​=(n−i)/n∗(s−j)/s。
分别有如下转移：f[i][j]=p1∗f[i][j]+p2∗f[i][j+1]+p3∗f[i+1][j]+p4∗f[i+1][j+1]f[i][j] = p_1 * f[i][j] +p_2*f[i][j+1] + p_3*f[i+1][j] + p_4*f[i+1][j+1]f[i][j]=p1​∗f[i][j]+p2​∗f[i][j+1]+p3​∗f[i+1][j]+p4​∗f[i+1][j+1]。
移项：(1−p1)∗f[i][j]=p2∗f[i][j+1]+p3∗f[i+1][j]+p4∗f[i+1][j+1](1-p_1)*f[i][j] =p_2*f[i][j+1] + p_3*f[i+1][j] + p_4*f[i+1][j+1](1−p1​)∗f[i][j]=p2​∗f[i][j+1]+p3​∗f[i+1][j]+p4​∗f[i+1][j+1];

f[i][j]=(p2∗f[i][j+1]+p3∗f[i+1][j]+p4∗f[i+1][j+1])/(1−p1)f[i][j] =(p_2*f[i][j+1] + p_3*f[i+1][j] + p_4*f[i+1][j+1]) / (1-p_1)f[i][j]=(p2​∗f[i][j+1]+p3​∗f[i+1][j]+p4​∗f[i+1][j+1])/(1−p1​)。
